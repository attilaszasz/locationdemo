Pros

  • No need for user interaction
  • Free
  • Browser agnostic
  • If the usage limit is a problem, we can run our own instance of this system. It's open source and freely available at GitHub.

Cons

  • API usage is limited to 10,000 queries per hour. After reaching this limit, all requests will result in HTTP 403
  • Uses the free (and not very accurate) database from MaxMind.
  • Every time there is a costly web request (unless we run our own instance locally)
