Pros

  • Standard Html5
  • No need for an expensive web api call
  • Free
  • Unlimited requests

Cons

  • Detection happens clientside, so in order to customize content, we have to make a redirection or an ajax call
  • IE9+ (but there are fallbacks)
  • Needs explicit consent from user
  • User may have concerns about sharing location
  • Every browser implements permission request differently
  • Only as accurate as the underlying technology, which is different from device to device
